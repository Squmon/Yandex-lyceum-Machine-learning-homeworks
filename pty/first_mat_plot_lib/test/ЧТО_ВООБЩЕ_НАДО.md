N = общее коичество

### доля правильных ответов
accaracy = (TP + TN)/(TP + TN + FP + FN) = (TP + TN)/N - аккуратность

### доля правильных положительных от всех положительных
Precision = TP/(TP + FP) - точность 

### на сколько правильно модель ответила
Recall = TP/(TP + FN) - Полнота


F-мера = 2/(1/())
