# Yandex Lyceum: Machine Learning Course Homeworks

Complete coursework repository for the Machine Learning program at Yandex Lyceum, demonstrating a comprehensive progression from fundamental data visualization through deep learning and neural networks.

## Overview

This repository contains coursework spanning 6 major topics in machine learning and data science, with implementations ranging from foundational concepts to advanced deep learning techniques. Each section builds upon previous knowledge to create a cohesive learning pathway.

## How to Use This Repository

### Prerequisites
- Python 3.7+
- pip package manager
- Jupyter Notebook

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/yandex-lyceum-ml.git
cd yandex-lyceum-ml

# Install required packages
pip install numpy pandas matplotlib seaborn scikit-learn pytorch jupyter

# Launch Jupyter
jupyter notebook
```

### Running Assignments

Each module contains Jupyter notebooks (.ipynb files) that can be executed:

```bash
jupyter notebook 01_visualization/matplotlib_basics/prog1.ipynb
```

All notebooks include:
- Problem statement and requirements
- Sample data and datasets
- Step-by-step solution implementation
- Visualizations and results
- Comments and explanations
